12-skills-tab · data sources

  1. Tab nav ← static HTML, active tab highlighted. Each tab is its own file: 12-project-config.html (contract), 12-skills-tab.html (this), 12-hooks-tab.html, 12-phase-defaults-tab.html.
  2. Cost impact summary ← aggregate: SUM(skill.cost_impact_per_call) × est call rate. Context overhead = SUM(skill.context_token_size) / avg call size.
  3. Scrutiny ← static (template requirement). Points to sibling tabs.
  4. Attached skills list ← project_skill WHERE project_id=acme-erp-migration JOIN skill. Attachment column = derived from skill.auto_attach_rule + project_skill.override_mode.
  5. Actions — "Edit" → 11-skill-edit.html?id={skill_id}&project=acme-erp-migration. "Detach" → POST /project/acme-erp-migration/skill/{id}/detach, with confirmation since detach affects future tasks. "Enable" (for disabled hooks) → redirect to 12-hooks-tab.html.
Destinations: "+ Add from library" → 11-skills-library.html?mode=install&project=acme-erp-migration. Rebalance → opens sidebar suggestions panel (AI sidebar mockup 16).
